# devup — Linux Software Version Visualizer

**devup** — это простая утилита для Linux. Она проверяет версии важных инструментов разработчика (GCC, Python, Git, Curl, ядро Linux) и показывает, устарели они или нет.

Все данные выводятся в виде цветной таблицы прямо в терминал.

---

## ?? Что понадобится

- Любой дистрибутив Linux (Ubuntu, Debian, Fedora — подойдет любой)
- Интернет для установки инструментов сборки
- Пара минут свободного времени
